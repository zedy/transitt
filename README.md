## Technologies Used
- NextJS (Framework)
- React Hook Form + Yup (Form controller + Validator)

### Additional Libraries
- Lodash (Utility)
- HotTostr (Notifications)
- DayJS (DateTime formatter)

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
